# 👁️ OptiCare — Eyewear Shop Manager

A free, cloud-based management system built for small optical shops with 1–2 branches.

## Features

- 🧾 **Billing** — Generate bills with prescriptions, send via WhatsApp, print
- 👤 **Customer Records** — Full history, prescriptions, spending
- 📦 **Stock Tracking** — Inventory with low-stock alerts
- 🔔 **Follow-Up Reminders** — Auto-list customers due for annual checkup
- 🏪 **Two Branches** — Switch between branches with one tap
- ☁️ **Firebase Backend** — Data safe in the cloud, never lost
- 📱 **Works on Phone** — Add to home screen, feels like a native app
- 💰 **Cost: ₹0 forever**

## Tech Stack

- Vanilla HTML + CSS + JavaScript (no build tools needed)
- Firebase Firestore (free tier)
- GitHub Pages (free hosting)
- Progressive Web App (PWA)
